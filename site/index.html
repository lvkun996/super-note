Windows 桌面笔记

Super Note

多栏画板、独立视图和全局搜索，让信息整理保持直接。

Windows 10 / 11 Windows 7 / 8

核心能力

一个画板，按需要展开

01

自由画板

放置文字与图片，移动、缩放并保持内容清晰。

02

动态多栏

从当前视图向左或向右继续分栏，各栏独立操作。

03

全局搜索

匹配画板与文件内容，点击结果即可定位。

04

本地优先

直接打开和保存本地文件，工作区自动保留。

  1. v0.1.4 2026.06.30

    项目网站与直接下载

    重构项目页面视觉与响应式布局，安装包可从网站直接下载。

    多栏工作区

    加入自由画板、动态分栏、全局搜索与本地工作区保存。
