Windows 桌面笔记

Super Note

多栏画板、独立视图和全局搜索，让信息整理保持直接。

Windows 10 / 11 Windows 7 / 8（v0.1.7）

核心能力

一个画板，按需要展开

01

自由画板

放置文字与图片，移动、缩放并保持内容清晰。

02

动态多栏

从当前视图向左或向右继续分栏，各栏独立操作。

03

全局搜索

匹配画板与文件内容，点击结果即可定位。

04

本地优先

直接打开和保存本地文件，工作区自动保留。

版本更新

每一次迭代，都有迹可循

简洁记录 Super Note 的功能变化与体验改进。

  1. v0.1.8 2026.07.17

    链接、搜索与编辑体验

    修复文本底部点击和搜索定位，新增 Ctrl + 单击外部链接、固定编辑菜单，并按功能拆分核心代码。

  2. v0.1.7 2026.07.09

    透明图标与作者全屏

    更新透明背景的新 logo，作者寄语改为全屏展示，优化标签栏边框，并暂时移除文本对比插件。

  3. v0.1.6 2026.07.08

    自动更新与全局快捷键

    根据凯哥的提议，新增自动更新按钮、Windows 版本通道识别、一键发布脚本，以及 Ctrl + Alt + 空格全局打开/隐藏。

  4. v0.1.5 2026.07.08

    为凯哥更新的版本

    新增凯哥生日快乐独立页面，用纯 CSS 做蛋糕、礼物、烟花与生日签互动；同时保留空工作区、插件开关、安装目录选择和本地打包缓存优化。

  5. v0.1.4 2026.06.30

    文本模块字号与滚动空间

    新增文本模块字号放大、缩小快捷键设置，并把文本编辑区底部空白扩展到约 22 行。

  6. v0.1.3 2026.06.24

    更多 Windows 版本支持

    新增 Windows 7 / 8 安装包，同时保留 Windows 10 / 11 独立下载入口。

  7. v0.1.2 2026.06.23

    项目网站与直接下载

    重构项目页面视觉与响应式布局，安装包可从网站直接下载。

  8. v0.1.1 2026.06.22

    多栏工作区

    加入自由画板、动态分栏、全局搜索与本地工作区保存。
